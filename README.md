# Projeto brain.md

Este repositório contém minhas anotações sobre diversos assuntos, principalmente relacionados à tecnologia da informação. Consiste em uma série de arquivos markdown, que podem estar interligados entre si diretamente ou por tags.

Os arquivos markdown foram escritos utilizando o aplicativo Obsidian, e portanto, faz o uso de sintaxes disponíveis nesta aplicação para realizar associações entre os arquivos para geração de grafos.
